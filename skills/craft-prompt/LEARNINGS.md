# craft-prompt — learnings queue

Continuous captures from `/craft-prompt` runs (see SKILL.md → Learning) — appended inline the moment a
run goes off the documented playbook, backed by a concrete artifact. Promotion is handled by `/learn`:
it drains entries below the marker, re-verifies each still holds, applies to the routed destination,
stamps `(since <date> · <artifact>)`, and checks the entry off.

Format: `- [ ] (YYYY-MM-DD · <artifact>) → <dest>: <one-line change>`
Destinations: **SKILL.md** · **references/techniques.md** · **references/claude-code/exemplars.md** · **references/playbook.md** · **the repo** (`.claude/`) · **user memory**.
