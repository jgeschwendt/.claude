Added 2026-07-12 from a heavy-mode self-audit run (audit workspace `skills/gigaresearch/workspaces/research-skill-improvement`, personal machine only — the primary sources below are the portable record); the seven load-bearing claims were verified blind against primary texts that day. Magnitude caveat: several headline numbers are LLM-judged win rates, and instance-level judge–human agreement on long-form output is weak (arXiv 2603.06942) — directions below are multi-source; treat magnitudes as soft.

- **Living outline steering the next wave** — Salesforce "Don't Stop Early" (arXiv 2604.24978): removing outline generation was the largest ablation drop (HAA 82.09→67.30, coverage 4.31→3.80); TTD-DR (arXiv 2507.16075): draft-conditioned retrieval adds +8–15pp win rate over self-evolution alone, +12pp query novelty; WebWeaver (arXiv 2509.13312): outline optimization rounds lift comprehensiveness 48.85→50.82.
- **Section-wise writing from per-section notes** — WebWeaver ablation: hierarchical writing vs full-context dump: insight 42.72→50.02, citation accuracy 86.73→93.37; LLM×MapReduce-V2 (arXiv 2504.05732): full-dump 25% claim recall, outline-only collapses claim density (135 vs 475); Chroma context-rot (2025-07): focused contexts beat full contexts by up to 60+pp.
- **Link pass** — arXiv 2604.03173: deployed agents emit 3–13% hallucinated URLs, 5–18% non-resolving; deterministic liveness check + one self-correction pass → under 1% (6–79×, p<10⁻³⁵). Support-checkers silently pass fabricated URLs.
- **Precision audit / fabrication as top failure** — DEFT (arXiv 2512.01948): strategic content fabrication is the largest of 14 failure modes at 18.95%; generation 38.76% > retrieval 33.10%; DeepWeb-Bench (arXiv 2605.21482): strong models fail via hallucinated precision (38%); BrowseComp (arXiv 2504.12516): browsing tools worsen calibration.
- **Retrieval depth degrades synthesis** — "Cited but Not Verified" (arXiv 2605.06635): fact-level citation accuracy −42% scaling 2→150 tool calls while link validity stays >94%; arXiv 2602.01672: over-retrieval degrades accuracy, adaptive control recovers it with fewer searches.
- **Evidence-aware, two-sided stopping** — Salesforce ablation: −8.4 HAA without evidence-aware termination (agents stop on surface evidence); arXiv 2606.20724: premature termination on partial answers (FM2) and near-duplicate query loops (FM1) sit in a failure family covering 71% of imperfect episodes; production consensus (Anthropic, LangChain, Kimi): the pre-fix failure was over-searching — stopping is always engineered.
- **Brief boundaries** — Anthropic multi-agent post (2025-06): vague subagent briefs measurably caused duplicated, divergent work; briefs need objective/format/tools/boundaries; token usage explains 80% of browsing-performance variance; claude.com (2026-01): decompose by context boundaries, not task type.
- **Broad-short-first querying** — Anthropic: untuned agents default to over-specific queries that return few results; broad-then-narrow fixed it.
- **Viewpoint-conditioned synthesis** — Diverge (arXiv 2602.00238, ICML 2025): retrieval diversity alone does not survive into answers (models homogenize); viewpoint-conditioning restores 2.1–2.7× semantic diversity at ~0.04/5 quality cost.
- **Corroborated as-designed** (no change needed): claims ledger with contradictions/gaps (Dossier, ACM CAIS 2026: +23pp BrowseComp-Plus vs ReAct); compression at every stage boundary and schema-forced subagent returns (universal across production systems); restorable-over-lossy compression — keep the URL, drop the page (Manus 2025-07); scripted resume protocol (Anthropic harnesses 2025-11).

Added 2026-07-14 from the skill-gap-analysis pass (wave-2 verified against primaries; audit workspace `skills/gigaresearch/workspaces/skill-gap-analysis-2026`, personal machine only):

- **VoI action selection** — Inference-Time Budget Control (arXiv 2605.05701): ranking Search/Decompose/Answer by marginal value per remaining budget: relative F1 +5.7–18.4% across four multi-hop benchmarks, −27.2% wall time; gains concentrate at low/mid budgets.
- **Retriever quality over query volume** — BrowseComp-Plus (arXiv 2508.06600, ACL 2026): GPT-5 55.9→70.1% from a retriever swap, with fewer search calls; compensating for weak retrieval with more queries is measurably inferior.
- **PDF/figure integrity is the bottleneck** — MMDR-Bench (arXiv 2601.12346): text-only pipelines systematically lose on citation-evidence and text-visual integrity; agents recall <5% of expert-core papers (arXiv 2601.12369).
- **GraphRAG rejected for open-web runs** — GraphRAG-Bench (arXiv 2506.05690, ICLR 2026): ~368× token cost, −11.6pts on simple lookup, fixed-corpus assumption; the ledger + section-wise synthesis already approximates its multi-hop benefit.
- **Causal memory for standing research** — AMA-Bench (arXiv 2602.22769): causality-linked memory +11.2pts over similarity recall; encoded in unattended-mode recurring-runs.
